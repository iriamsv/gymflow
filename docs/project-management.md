# Gestión del proyecto

Para organizar el desarrollo de GymFlow se ha utilizado Trello siguiendo una metodología inspirada en Kanban.

## Columnas utilizadas

* Backlog: funcionalidades pendientes.
* Todo: tareas preparadas para comenzar.
* In Progress: tareas en desarrollo.
* Review: tareas finalizadas pendientes de revisión.
* Done: tareas completadas.

## Organización

Cada funcionalidad principal del proyecto se ha representado mediante una tarjeta independiente. Las funcionalidades complejas se han dividido en subtareas mediante checklists para facilitar el seguimiento del progreso.

El tablero permite visualizar el estado actual del proyecto y priorizar las tareas pendientes.
