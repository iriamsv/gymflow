# Retrospectiva del proyecto

## Qué he aprendido

Durante el desarrollo de GymFlow he aprendido a construir una aplicación fullstack utilizando tecnologías modernas tanto en frontend como en backend.

En el frontend he trabajado con React, TypeScript, Tailwind CSS, React Router, Context API y distintos hooks como useState, useEffect, useMemo y useCallback.

En el backend he desarrollado una API REST utilizando Node.js y Express siguiendo una arquitectura por capas basada en routes, controllers y services.

También he aprendido a conectar el frontend con el backend mediante un cliente API tipado y a gestionar estados de carga, éxito y error.

## Integración entre frontend y backend

Uno de los aspectos más importantes del proyecto ha sido la integración entre React y Express.

El frontend consume los datos proporcionados por la API mediante peticiones HTTP y utiliza Context API para compartir la información entre los distintos componentes.

La API actúa como fuente de verdad de los datos de la aplicación.

## Problemas encontrados

Durante el desarrollo aparecieron varios problemas:

* Configuración inicial de TypeScript.
* Conexión entre React y Express.
* Gestión del estado global mediante Context API.
* Manejo de errores en las peticiones HTTP.
* Configuración de rutas y estructura del proyecto.

Cada problema se fue resolviendo mediante pruebas, depuración y revisión del código.

## Uso de inteligencia artificial

La inteligencia artificial se utilizó como herramienta de apoyo para:

* Resolver errores de programación.
* Comprender conceptos de React y TypeScript.
* Diseñar la arquitectura de la aplicación.
* Generar ejemplos de código.

Las soluciones propuestas fueron revisadas y adaptadas antes de incorporarlas al proyecto.

## Valoración personal

Este proyecto me ha permitido comprender mejor cómo se desarrolla una aplicación fullstack moderna.

He podido aplicar conocimientos de frontend, backend, APIs REST, TypeScript y organización de proyectos en un único desarrollo práctico.

Además, he mejorado mis habilidades de resolución de problemas, documentación y uso de herramientas de desarrollo.
