# Sistema de rutas

GymFlow utiliza React Router para la navegación entre páginas.

## Rutas implementadas

### Dashboard

```text
/
```

Página principal de la aplicación.

### Rutinas

```text
/ workouts
```

Muestra las rutinas disponibles y permite crear nuevas.

### Estadísticas

```text
/ stats
```

Muestra estadísticas calculadas a partir de las rutinas.

### Página 404

```text
*
```

Se muestra cuando el usuario accede a una ruta inexistente.

## Ventajas

* Navegación sin recarga de página.
* Mejor experiencia de usuario.
* Separación clara de funcionalidades.
