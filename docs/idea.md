# Idea del proyecto: GymFlow

## Descripción

GymFlow es una aplicación web para gestionar entrenamientos de gimnasio y realizar seguimiento del progreso deportivo.

## Problema que resuelve

Muchas personas organizan sus entrenamientos mediante notas, papel o aplicaciones poco personalizadas. GymFlow centraliza toda la información relacionada con rutinas y entrenamientos en una única plataforma.

## Usuario objetivo

Personas que entrenan regularmente en gimnasio y desean organizar mejor sus rutinas y objetivos.

## Funcionalidades principales

* Crear rutinas de entrenamiento.
* Gestionar ejercicios.
* Registrar entrenamientos realizados.
* Marcar ejercicios como favoritos.
* Visualizar estadísticas básicas.
* Calendario de entrenamientos.

## Funcionalidades opcionales

* Modo oscuro.
* Objetivos semanales.
* Seguimiento de pesos levantados.
* Gráficas de progreso.

## Mejoras futuras

* Sistema de usuarios.
* Sincronización en la nube.
* Compartir rutinas.
* Integración con dispositivos deportivos.
