# Despliegue

## Frontend

El frontend se desplegó utilizando Vercel.

Proceso seguido:

1. Subir el proyecto a GitHub.
2. Conectar el repositorio con Vercel.
3. Configurar el proyecto Vite.
4. Ejecutar el despliegue automático.

## Backend

Durante el desarrollo el backend se ejecutó localmente mediante Express.

Para producción se podría desplegar en:

* Vercel
* Render
* Railway

## Verificación

Se comprobó que:

* La aplicación se despliega correctamente.
* Las rutas funcionan.
* La interfaz se muestra correctamente.
